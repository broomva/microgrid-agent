### Failure Isolation Between Planes

```
If Plane 3 dies (cloud/fleet):
  вҶ’ Plane 2 continues (local ML, local learning)
  вҶ’ Plane 1 continues (dispatch, safety)
  вҶ’ Lights stay on вң“

If Plane 2 dies (Python ML worker):
  вҶ’ Plane 1 continues with last-known-good forecast
  вҶ’ Falls back to persistence prediction (yesterday=today)
  вҶ’ Falls back to rule-based dispatch
  вҶ’ Lights stay on вң“

If Plane 1 dies (Rust kernel):
  вҶ’ systemd restarts in 5s (WatchdogSec=30s)
  вҶ’ If 4 crashes in 3 min вҶ’ full RPi reboot
  вҶ’ During restart: last dispatch commands hold (inverter keeps state)
  вҶ’ Lights stay on вң“ (briefly degraded)

If Plane 0 dies (hardware failure):
  вҶ’ Kernel detects via sensor timeout
  вҶ’ Flags anomaly, notifies fleet
  вҶ’ Requires physical maintenance
  вҶ’ This is the only failure that affects power
```
